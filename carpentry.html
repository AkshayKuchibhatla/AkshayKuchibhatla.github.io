Beveled-Top Sawhorse
Project was developed for STEM Exhibition. Task was to develop an innovative communication device and demonstrate by sending signals accross a distance of few feet.
Description: A laser light beam was sent to a receiver which senses the light. The light is sent from two emitters as ON & OFF. if ON there will be light if OFF there will be no light sent. The sensor senses the pulse and decodes it into binary 1s and 0s. The binary is converted to Asci vlaues of alphabets and translated to text.
Hardware: 2 Arduino Unos, Laser Light emitters, laser light sensors and LCD display to display the message text.
Programming Language & Tools: C, Arduino IDE.
Adirondack Chair
Project was developed for watering plants in a Home Garden Task is to develop a system to invoke a water pump from a web page.
Description: A web page with buttons on it to invoke water pumps upon button click. The button click sends a HTTP Request to a ESP8266 based system. the ESP8266 will send signals to a relay switch to switch on and off a water pump.
Hardware: ESP8266 (Arduino Compatible), RaspberryPi to host the web page, Relay Switch.
Programming Language & Tools: C, Arduino IDE, HTML, JavaScript, VSCode.
